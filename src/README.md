## Descripción

Este proyecto implementa un sistema de partículas en tiempo real utilizando shaders (vertex y fragment) en GPU.  

Se desarrollaron dos comportamientos distintos:

-  **Fountain**: Simulación de partículas basada en física (gravedad y viento).
-  **Galaxy**: Sistema de partículas con movimiento orbital en espiral.

Ambos efectos son interactivos y permiten modificar parámetros en tiempo real.
